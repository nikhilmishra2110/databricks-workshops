# Prompt Library

Prompt cards in this folder are meant to be copied into workshop-specific prompt packs, adapted, and versioned.

Each prompt should include:

- purpose
- best used when
- inputs
- prompt text
- expected output
- human review checklist

Use placeholders in double braces:

```text
{{workshop_goal}}
{{audience}}
{{database_dialect}}
{{time_limit_minutes}}
```
