# Contributing

This repo should stay practical, reusable, and easy to scan during workshop prep.

## What Belongs Here

- reusable prompt cards
- workshop agendas, facilitator guides, learner handouts, and exercises
- SQL setup scripts, seed data, and sample schemas
- readiness checklists, rubrics, and retrospectives
- notes that help future facilitators run the same workshop better

## What Does Not Belong Here

- secrets, credentials, API keys, or connection strings
- private customer data
- recordings, transcripts, or participant names without explicit approval
- very large generated exports

## Pull Request Checklist

Before opening a pull request:

- [ ] the asset has a clear title, audience, and intended outcome
- [ ] prompts include inputs, expected outputs, and review guidance
- [ ] SQL examples are runnable or clearly marked as pseudocode
- [ ] workshop materials avoid sensitive data
- [ ] docs link to related assets where useful

## Review Standards

Reviewers should focus on:

- workshop clarity
- database correctness
- prompt specificity
- participant safety and privacy
- reuse across future workshops
