## Before The Workshop

- [ ] Confirm environment setup
- [ ] Run every setup script
- [ ] Test each solution query
- [ ] Prepare fallback screenshots or query results
- [ ] Review participant prerequisites

## Opening

State the workshop goal, the expected working style, and what participants will produce.

## Facilitation Notes

| Moment | Watch For | Suggested Intervention |
| --- | --- | --- |
| Setup | Missing database client or permissions | Pair participants or use shared fallback |
| First query | Confusion about table names | Point to schema context before giving syntax |
| Joins | Joining on the wrong key | Ask what one row in each table represents |
| Aggregation | Filtering before vs after grouping | Ask whether the condition applies to rows or groups |

## Common Questions

### What if participants finish early?

Use the stretch tasks in each exercise.

### What if setup fails?

Use the provided query results and focus on reading, debugging, and explaining SQL.

## Closing

- Ask participants to name one query pattern they can use this week.
- Capture confusing points in `workshop-retrospective.md` after the session.
