# Workshop Template

Copy this folder when creating a new workshop.

```bash
cp -R workshops/_template workshops/{{workshop-slug}}
```

Recommended workshop folder structure:

```text
workshops/{{workshop-slug}}/
|-- workshop-plan.md
|-- agenda.md
|-- facilitator-guide.md
|-- learner-handout.md
|-- setup.md
|-- exercises/
|-- prompts/
`-- assets/
```

Keep each workshop self-contained, but link back to reusable prompts and rubrics when possible.
