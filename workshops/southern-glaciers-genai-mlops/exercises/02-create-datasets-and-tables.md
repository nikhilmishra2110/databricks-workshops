# Exercise 2: Create Datasets And Tables

## Goal

Use Genie Code to generate synthetic data and create Unity Catalog tables.

## Steps

1. Open `../prompts/prompt-pack.md`.
2. Use prompt 1, `Create Synthetic Datasets And Tables`.
3. Review the plan before accepting code.
4. Confirm the generated data matches `../assets/source-data-contract.yml`.
5. Run validation queries.

## Done When

- Bronze tables exist in Unity Catalog.
- Data is synthetic.
- Table and column comments are present.
- The row counts are small enough for a workshop.

## Review Questions

- Did the generated data include useful edge cases?
- Are there any accidental real-world names or identifiers?
- Did Genie Code choose reasonable data types?
