# 00 - Before The Workshop

## Goal

Make the workshop low effort on delivery day by confirming workspace readiness before participants arrive.

## Use Genie Code Mode

Use Genie Code Agent mode if available. Use plan-only mode if you do not want it to inspect or create anything.

## Done When

- Skills are treated as optional; prompt-only delivery is confirmed.
- Unity Catalog permissions are confirmed.
- SQL warehouse access is confirmed.
- Lakeflow pipeline permissions are confirmed.
- Genie spaces are available.
- Metric views are available.
- MLflow experiment access is confirmed.
- Lakebase and Databricks Apps are confirmed or marked as backup-only.
- Backup notebooks are available.

## Backup

If a feature is unavailable, mark it in the readiness report and use `../99-backup/manual-runbook.md`.
