# Exercise 3: Build Lakeflow Pipeline

## Goal

Use Genie Code Agent mode to generate a Lakeflow Spark Declarative Pipeline.

## Steps

1. Open the Lakeflow Pipelines Editor.
2. Use prompt 2 from `../prompts/prompt-pack.md`.
3. Ask Genie Code for the design first.
4. Review source tables, output tables, expectations, and data quality checks.
5. Accept or paste the generated pipeline code.
6. Validate the pipeline and inspect errors.
7. Ask Genie Code to fix one error or improve one expectation.

## Done When

- Pipeline source code is generated.
- Outputs are defined for silver and gold tables.
- Expectations cover IDs, statuses, amounts, and inventory math.
- The team can explain streaming table vs materialized view choices.
