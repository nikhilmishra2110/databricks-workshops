# Exercise 5: MLflow GenAI/MLOps

## Goal

Track and evaluate a GenAI recommendation workflow.

## Steps

1. Use prompt 4 from `../prompts/prompt-pack.md`.
2. Create an evaluation dataset with synthetic account scenarios.
3. Compare two prompt variants for next-best-action recommendations.
4. Log inputs, outputs, prompt versions, metrics, and artifacts.
5. Review failures and improve one prompt.

## Done When

- There is an MLflow experiment for the recommendation workflow.
- Test cases include normal, edge, and failure scenarios.
- Results can be compared across prompt versions.
- Human feedback fields are defined.

Use `../assets/mlflow-evaluation-rubric.yml`.
