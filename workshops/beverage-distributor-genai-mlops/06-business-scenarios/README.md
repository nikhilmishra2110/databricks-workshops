# 06 - Business Scenarios

## Goal

Make the Genie space feel real by having business users ask natural-language questions in a scenario.

## Scenario

A regional sales leader is preparing for a weekly account review. They want to know which accounts need follow-up, which reps need help, and where inventory or promotion issues affect action plans.

## Done When

- Participants run at least six business questions.
- The group compares trusted answers, weak answers, and missing context.
- Improvements are fed back into the Genie space.

## Backup

If Genie is unavailable, run the scenario with SQL queries from `../99-backup/fallback-genie-questions.md`.
