# 01 - Use Case

## Goal

Start with one use case: Field Sales Account Intelligence Copilot.

## Business Story

Sales leaders and field reps need to know which accounts need attention, why they need attention, and what action should happen next.

## Done When

- The use case is written in business language.
- Participants know the user, decision, data, action, and success criteria.
- The target Databricks architecture is visible.
