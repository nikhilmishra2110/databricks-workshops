# 07 - MLflow GenAI Evaluation

## Goal

Show how Databricks supports MLOps for GenAI recommendations with MLflow tracking, traces, evaluation examples, and feedback.

## What Gets Evaluated

The next-best-action recommendation for an account.

## Done When

- MLflow experiment is created.
- Evaluation dataset exists.
- Two prompt variants are compared.
- Results include quality dimensions and failure analysis.
