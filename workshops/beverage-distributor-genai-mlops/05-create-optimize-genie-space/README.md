# 05 - Create And Optimize Genie Space

## Goal

Create a Genie space that lets business users ask trusted questions in natural language.

## Included Assets

- `workshop.beverage_distribution.beverage_account_metrics`
- `workshop.beverage_distribution.gold_account_intelligence`
- `workshop.beverage_distribution.silver_account_orders`
- `workshop.beverage_distribution.silver_inventory_position`

## Done When

- Genie space is created or a precise manual creation plan is produced.
- Curated tables and the metric view are included.
- Business terms are defined.
- Sample questions are loaded or documented.
- Weak-answer risks are captured.

## Backup

If Genie space creation is unavailable, use `../99-backup/fallback-genie-questions.md` as a demo script in SQL or dashboards.
