# Pre-Workshop Checklist

## Two Weeks Before

- [ ] Confirm audience and skill level
- [ ] Confirm database dialect and tool access
- [ ] Choose workshop package or create one from `workshops/_template`
- [ ] Identify sensitive-data constraints
- [ ] Draft learning outcomes

## One Week Before

- [ ] Test setup instructions on a clean machine or environment
- [ ] Run all setup SQL
- [ ] Run all solution queries
- [ ] Review facilitator notes
- [ ] Prepare fallback plan for setup failures

## Day Before

- [ ] Confirm participant instructions were sent
- [ ] Re-run setup verification query
- [ ] Open all files needed during facilitation
- [ ] Prepare timing checkpoints

## After

- [ ] Capture participant feedback
- [ ] Update confusing instructions
- [ ] Open issues for follow-up fixes
- [ ] Promote reusable assets back into shared folders
